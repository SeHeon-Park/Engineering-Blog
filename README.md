## Description
Spring Boot + JPA로 구현한 과목별로 내용을 정리 해놓을 수 있는 기술 블로그

## 프로젝트 정보
### URL
http://3.39.184.90:8080/

### 개발 스택
- Spring Framework
- Spring Security
- JPA
- MySql
- Thymleaf
